O TechFlow Mobile é o aplicativo oficial do ecossistema TechFlow, criado para facilitar a abertura, consulta e acompanhamento de chamados técnicos.
Ele foi desenvolvido especialmente para técnicos e usuários que precisam de agilidade no dia a dia, oferecendo uma interface simples, direta e totalmente integrada à API do sistema.

O objetivo é proporcionar uma experiência rápida e prática, conectando colaboradores e equipe técnica de forma eficiente.

O app oferece 
Login e autenticação integrada com a API

1 - Abertura de novos chamados diretamente pelo celular

2 - Consulta dos chamados existentes

3 - Listagem de chamados em aberto, finalizados e históricos

4 - Acesso ao FAQ com dúvidas comuns

5 - Interface moderna e responsiva

6 - Comunicação direta com o servidor via Retrofit

7 - Integração completa com o banco de dados Azure SQL

8 - Suporte a perfis: Usuário e Técnico

Tecnologias Utilizadas

Android Studio

Java

XML (layouts) - Retrofit2 + Gson - OkHttp - API Rest ASP.NET - SQL Server / Azure SQL - MVVM (estrutura recomendada) - RecyclerView e Adapter personalizados


***********************************************************COMO EXECUTAR O PROJETO *******************************************************************
1 - Faça o clone do repositório:
git clone https://github.com/Vieirafps/TechFlow-Mobile.git

2 - Abra no Android Studio
3 - Aguarde o Gradle sincronizar automaticamente
4 - Configure a URL base da API no arquivo apropriado
5 - Execute o aplicativo em um emulador ou dispositivo físico
